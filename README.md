# Multi-Tenant SaaS Platform

A production-ready multi-tenant SaaS application for managing projects and tasks with role-based access control and subscription management.

## Features
- Multi-tenancy with data isolation
- JWT authentication
- Role-based access control
- Subscription plans
- Dockerized deployment

## Tech Stack
- Node.js
- Express
- PostgreSQL
- React
- Docker

## Setup
docker-compose up -d
